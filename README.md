Yelp Search Engine
==================

Search engine for the [Yelp dataset](https://www.yelp.com/dataset/challenge)
implemented with [Lucene](https://lucene.apache.org/core/) in Scala.

## Usage
Run the importer:

```
sbt
> runMain ir.Importer
```

Run the webapp:

```
> jetty:start
```
